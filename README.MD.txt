# Smart Cyber Threat Detection and Monitoring System

Project Description
This project is a Python-based system designed to monitor system activities and detect potential cyber threats such as brute force attacks.

Features
- User authentication
- Real-time activity monitoring
- Threat detection (rule-based)
- Alert generation
- Logging system
- Threat storage
- Unit & Integration testing

Architecture
The system follows a layered architecture:
- Presentation Layer (User interaction)
- Business Logic Layer (Detection engine)
- Data Layer (Logs & storage)

Technologies Used
- Python
- File Handling (Logs & Threats)
- unittest (Testing)

How to Run

```bash
cd src
python main.py
